# Grading System Website

## Overview
This is a simple web application designed for managing student grades. It allows you to view a list of students, enter grades for individual students, and view a summary of all grades.

## Features
- View a list of students.
- Enter and update grades for each student. To update the grades of a given student just resubmit the newly updated grades to the grade submission page, always make sure that you selected the student you want to enter the grades for first.
- Calculate and display average grades for homework, quizzes, midterm, final project, and overall final grade. All grade submissions will be displayed on the last page, with the latest submission on the bottom.


## Requirements
- PHP
- MySQL 
- XAMPP or any other local server environment.
